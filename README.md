# MenuPicker 사용 방법

> https://menu-picker-fe.vercel.app

## 웹
1. 링크에 접속한다.
2. 웹에서 바로 사용하거나, 홈화면 상단에 "앱으로 열기" 버튼을 누르면 QR이 뜬다.
3. 핸드폰으로 해당 QR을 찍으면 링크로 이동하고, 우측 상단에 다운로드 버튼이 표시되고, 클릭하여 기기에 설치해 사용할 수 있다.

# 모바일
1. 링크에 **크롬**으로 접속한다.
2. 바로 사용 하거나, 우측 상단에 다룬로트 버튼을 클릭하여 기기에 설치하여 쓸 수 있다.
